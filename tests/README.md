# Tests Directory

This directory contains cross-package tests, contract tests, integration tests, end-to-end tests, fixtures, and static verification inputs.

## Structure

```
tests/
  contract/      # Contract tests
  integration/   # Integration tests
  e2e/           # End-to-end tests
  fixtures/      # Test fixtures
  static/        # Static verification inputs
```

## Standards

- `TEST_SPEC.md`: Verification and contract testing expectations
