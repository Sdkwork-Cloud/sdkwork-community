# Configs Directory

This directory contains source-controlled safe config templates, profile examples, config schemas, and non-secret runtime defaults.

## Structure

```
configs/
  schemas/       # Config schemas
  profiles/      # Profile examples
  examples/      # Example configs
  defaults/      # Non-secret defaults
```

## Standards

- `CONFIG_SPEC.md`: Typed runtime config and SDK client initialization
- `ENVIRONMENT_SPEC.md`: Environment variables and runtime config files
