# SDKWork Community

`sdkwork-community` owns SDKWork community posts, discussions, questions, resources, public feeds, moderation state, recommendations, local Rust implementation, OpenAPI authority documents, and generated SDK families.

## Verification

```powershell
pnpm install
pnpm verify
```

## API Surfaces

- Open API: `/community/v3/api`
- App API: `/app/v3/api/community`
- Backend API: `/backend/v3/api/community`

Generated SDK families live under `sdks/sdkwork-community-sdk`, `sdks/sdkwork-community-app-sdk`, and `sdks/sdkwork-community-backend-sdk`.
